# Intro

My Advent of Code 2022 solutions. You need `bun` to use this repository.

# Commands

Run on real input

```bash
bun run dev [1-25]
```

Run on sample input

```bash
bun run sample [1-25]
```

Run performance tests on real input

```bash
bun run time ([1-25] | all)
```

Run correctness tests

```bash
bun run test [1-25]
```
